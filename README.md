# Customer Churn Data Cleaning and EDA

## Project Overview

This project was completed as part of the Codveda Data Science Internship.

The objective was to clean, preprocess, and analyze a customer churn dataset to identify patterns and prepare the data for machine learning.

## Tasks Performed

### Data Cleaning
- Missing value analysis
- Duplicate removal
- Label encoding
- Feature scaling
- Data preprocessing

### Exploratory Data Analysis
- Summary statistics
- Histograms
- Boxplots
- Correlation heatmap
- Churn distribution analysis

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
